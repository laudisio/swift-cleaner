## 🚀 Overview

Swift Cleaner is a lightweight Chrome extension designed to help developers and power users quickly manage browser data without navigating through Chrome settings.

With just one click, you can clear cookies, reset sessions, and perform a full cache bypass reload.

---

## ⚡ Features

* 🧹 **Clear ALL cookies instantly**
* 🌐 **Clear cookies for a specific website**
* 🧑‍💻 **Dev Mode: clear cache + hard reload (bypass cache)**
* 🔄 **Optional automatic site cookie removal in Dev Mode**
* 🎯 **Simple and minimal UI**
* ⚡ **Instant execution – no setup required**

---

## 🧠 Why Swift Cleaner?

When developing or testing applications, browser cache and cookies often cause:

* ❌ Old versions of JavaScript/CSS being loaded
* ❌ Session/login inconsistencies
* ❌ Debugging issues

Swift Cleaner solves this by providing a fast and reliable way to reset browser state.

---

## 🔥 Dev Mode Explained

Dev Mode performs:

1. Cache cleanup
2. (Optional) Cookie cleanup for the current site
3. Hard reload with cache bypass

Equivalent to:

```
Ctrl + Shift + R
```

But faster and automated.
